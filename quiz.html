Quiz Game

A web-based programming trivia quiz with 5 questions, navigation, score tracking, and a results screen — no frameworks, pure web.

Demo Video

🎬 Demo video coming soon

Project Overview

The Problem

Needed a fun interactive way to test programming knowledge, without any backend, server, or external libraries.

My Approach

Built entirely in vanilla HTML, CSS, and JavaScript. Questions render dynamically and the user can navigate back and forth freely.

Collaboration

Built together with my partner Hanane as a team project at AUT.

Result

A clean, responsive quiz app that runs entirely in the browser with zero dependencies.

Features

  • ❓ 5 Programming Questions — multiple choice trivia about code
  • ⬅️ Previous / Next Navigation — move freely between questions
  • 🏁 End Game Anytime — option to finish early
  • 🏆 Score Screen — results shown at the end with final score
  • 🔁 Restart — play again without refreshing the page
